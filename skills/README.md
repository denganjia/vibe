# Vibe Skills

This directory contains the skill definitions for AI agents using Vibe-CLI. These skills provide the necessary metadata, roles, SOPs, and templates for agents to operate effectively in the local development environment.

## Available Skills

### [Vibe-Operator](./vibe-operator/SKILL.md)
**Core System Skill**

The fundamental skill for multi-agent orchestration. It enables AI agents to:
- **Orchestrate**: Manage terminal panes and workspaces dynamically.
- **Synchronize State**: Maintain global awareness across multiple parallel agents.
- **Collaborate Safely**: Use human-in-the-loop approval flows for critical operations.

For detailed documentation, see the [Vibe-Operator Skill Definition](./vibe-operator/SKILL.md).
