# SDD Phase 1: Discussion

## Goal
Establish a clear understanding of the requirements and constraints with the human user.

## Instructions
1. **Analyze the Request**: Read the user's initial prompt carefully.
2. **Clarify Intent**: If the request is ambiguous, ask specific questions.
3. **Establish Constraints**:
   - Identify technical limitations.
   - Confirm desired tech stack (see `role.md` initialization).
4. **Report**: Use `vibe report` to log the initial alignment status.
5. **Outcome**: A verbal or written alignment on the scope of work.
