<task type="auto">
  <name>Task 3: 创建 MCP 自动化验证脚本</name>
  <files>plugin/vibe/scripts/test-mcp.js</files>
  <action>
创建 `plugin/vibe/scripts/test-mcp.js`，用于在不依赖外部 AI 客户端的情况下验证 MCP 服务器：
- 使用 `child_process.spawn` 启动 `node mcp-server.js`。
- 通过 stdin 发送 JSON-RPC `initialize` 请求。
- 发送 `tools/call` 请求调用 `vibe_ping`。
- 验证响应中包含 "pong"。
- 超时或收到异常输出时报错。
- 更新 `package.json` 中的 `scripts.test` 包含此验证步骤。
  </action>
