<objective>
Update the `.vibe` workspace templates to use a pure JSON schema, satisfying the plugin-first architecture requirements.

Purpose: Legacy markdown templates required complex YAML parsing. By migrating to pure JSON, the Phase 22 lightweight scripts can parse `.vibe/agents/*.json` natively using built-in `JSON.parse()`.
Output: Clean JSON agent templates and an updated nested `config.json`.
</objective>
